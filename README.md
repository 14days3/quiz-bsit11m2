# quiz-bsit11m2
this is a quiz made for our programming class meant to evaluate a person's understanding of python through basic python questions.
It's a 25 item test, it has a timer that records the amount of time spent within the program using the time module and it tracks the amount of correct and incorrect answers through the use of assignment operators.

The questions range from Python syntax, arithmetic operators, data types, Python's keywords and loop keywords.
It consists of 3 types of questions: Identification, Multiple choices and Enumeration.
